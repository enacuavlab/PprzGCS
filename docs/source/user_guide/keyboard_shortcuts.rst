Keyboard shortcuts
==================

+------------------------+---------------------------------------------------------------------------------------+
| Keyboard shortcuts     | Descriptions                                                                          |
+========================+=======================================================================================+
|r                       | Turn the map clockwise                                                                |
+------------------------+---------------------------------------------------------------------------------------+
|Ctrl+r                  | Turn the map counter-clockwise                                                        |
+------------------------+---------------------------------------------------------------------------------------+
|Alt+r                   | Reset the map orientation                                                             |
+------------------------+---------------------------------------------------------------------------------------+
|c                       | Center the map                                                                        |
+------------------------+---------------------------------------------------------------------------------------+
